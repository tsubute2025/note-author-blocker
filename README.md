# note-author-blocker

note.com で特定の作者の投稿を非表示にする Chrome 拡張機能です。

## 機能

- 指定した作者の投稿を「非表示」というプレースホルダーに置き換えます
- タイムライン、検索結果、カテゴリページなど、note.com 内のあらゆる場所で動作します
- 動的に読み込まれるコンテンツにも対応（無限スクロール等）

## note 公式のブロック機能との違い

note の公式ブロック機能では、ブロックしたユーザーの投稿が**検索結果には表示されてしまいます**。

この拡張機能は画面上で投稿を非表示にするため、検索結果を含むすべての場所で特定作者の投稿を見えなくできます。公式ブロック機能との併用がおすすめです。

## インストール方法

1. このリポジトリをダウンロードまたはクローン
2. Chrome で `chrome://extensions` を開く
3. 右上の「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択

## 使い方

### ブロックリストの設定

1. Chrome の拡張機能アイコンを右クリック
2. 「オプション」を選択
3. テキストエリアにブロックしたいユーザー ID を1行に1つずつ入力
4. 「保存」ボタンをクリック

### ユーザー ID の確認方法

ユーザー ID は、そのユーザーのプロフィールページの URL から確認できます。

```
https://note.com/example_user
                 ^^^^^^^^^^^^
                 この部分がユーザーID
```

## ファイル構成

```
├── manifest.json   # 拡張機能の設定ファイル
├── content.js      # 投稿を非表示にするメインスクリプト
├── options.html    # 設定画面の HTML
├── options.js      # 設定画面の処理
└── README.md
```

## 技術仕様

- Manifest V3 対応
- `chrome.storage.sync` を使用（Chrome アカウントで同期可能）
- MutationObserver による動的コンテンツへの対応

## 注意事項

- この拡張機能はクライアントサイドで動作するため、投稿データ自体は読み込まれます（通信量の削減にはなりません）
- note.com のDOM構造が変更された場合、正常に動作しなくなる可能性があります
